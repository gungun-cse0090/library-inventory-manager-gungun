## Overview

This is a **command-line Library Inventory Manager**.  
It helps track books in a campus library: add books, issue and return books, search catalog, and view all books.  
The project demonstrates **Object-Oriented Programming (OOP)** and simple file handling.

---

## Features

- Add new books (title, author, ISBN).  
- Issue and return books.  
- Search books by title or ISBN.  
- View all books with status (available / issued).  
- Persistent storage using a text file
- Menu-driven command-line interface.
